1 Limit

1.1 Definition

We say

limx→∞ f(x) = L

means that for each ε > 0, there exists a δ > 0 such that if

0 < |x - c| < δ, then |f(x) - L| < ε

1.2 Evaluating

1.2.1 Composition

Let {b c} : Real, n : Positive Int Let f, g be functions with the following limits:

limx→ c f(x) = L and limx→ c g(x) = K

  • limx→ c [bf(x)] = bL
  • limx→ c [f(x) ± g(x)] = L ± K
  • limx→ c [f(x)g(x)] = LK
  • limx→ c [f(x) / g(x)] = L / K
  • limx→ c [f(x)]n = Ln
  • limx→ c f(g(x)) = f(limx→ c g(x)) = f(K)
